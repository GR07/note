# MVC 和 MVVM

MVC 和 MVVM 不属于经典的 23 种设计模式，但也可以说他们是设计模式。

本来设计模式就是一种抽象的定义，而且随着时代的发展，它也需要慢慢的改变。

如何称呼无所谓，关键是理解它们的内容。



## MVC

MVC 原理

- View 传送指令到 Controller

- Controller 执行业务逻辑，修改 Model

- Model 变化重新渲染 View



## MVVM

MVVM 直接对标 Vue 即可

- View 即 Vue template

- Model 即 Vue data

- VM 即 Vue 其他核心功能，负责 View 和 Model 通讯
