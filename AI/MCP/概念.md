# 定义

Model Context Protocol
模型上下文协议

Anthropic 公司在 2024年11月25号发布的一个协议


# MCP 用途

大模型本身其实只会问答，并不会使用外部工具

MCP的出现就是让大模型拥有使用各种外部工具的能力


# MCP HOST

支持MCP协议的软件

借助它才能使用 MCP 调用外部工具 

常见的 MCP HOST

cloud desktop
cursor
cline - vscode 插件
cherry studio


# MCP Server 服务器

非传统的服务器（例：访问谷歌网站，即是访问谷歌服务器）

这里的服务器指的是，这个程序的执行是符合 MCP 协议的

MCP Server 都是本地通过 Node 或 Python 启动的（联网/不联网）

可以理解为就是一个应用软件内置了一些功能模块（例：手机应用），而功能模块就叫 Tool（工具函数）

例：一个获取天气的 MCP Server 里面可能包含 2 个 Tool（天气预告、气象预警）

回到现实：那么当我通过 MCP HOST 询问明天的天气时，MCP HOST 就可以安装这样一个天气的 MCP Server，并使用其中获取天气预告的 Tool，就能拿到明天的天气了。
